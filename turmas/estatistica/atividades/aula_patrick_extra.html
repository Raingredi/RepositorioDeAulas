Atividade — Manipulação de Listas em Python

  1. Questão 1 — Percorrendo uma lista

    Lista inicial: ["Ana", "Bruno", "Carlos"]

    • Use um for para exibir cada convidado com a mensagem: "Bem-vindo(a), X".
    • Depois, exiba também o total de convidados usando len().

  2. Questão 2 — Adicionando elementos

    Lista inicial: ["Ana", "Bruno", "Carlos"]

    • Adicione "Daniel" ao final da lista com .append().
    • Insira "Mariana" na posição 1 com .insert().
    • Mostre a lista final após essas modificações.

  3. Questão 3 — Removendo convidados

    Use a lista da questão anterior.

    • Remova o convidado "Carlos" usando .remove().
    • Remova o último elemento usando .pop().
    • Remova o convidado da posição 1 usando del.
    • Exiba a lista resultante.

  4. Questão 4 — Condições e remoções

    Lista inicial: ["Ana", "Carlos", "Clara", "Bruno", "Caio"]

    Remova todos os convidados cujo nome começa com a letra "C" usando um for e .remove(). Exiba a lista final.

  5. Questão 5 — Criando novas listas

    Lista inicial: ["Ana", "Bruno", "Daniel"]

    • Crie uma nova lista com os nomes em letras maiúsculas usando for ou list comprehension.
    • Exiba a lista original e a lista transformada.

  6. Questão 6 — Ordenação

    Lista inicial: ["Mariana", "Ana", "Carlos", "Bruno"]

    • Organize os nomes em ordem alfabética com .sort().
    • Depois, organize em ordem decrescente (Z → A).

  7. Questão 7 — Enumerando convidados

    Lista inicial: ["Ana", "Bruno", "Carlos"]

    Use enumerate() em um for para exibir os convidados numerados, no formato: "1º convidado: Ana".

  8. Questão 8 — Fatias de lista

    Lista inicial: ["Ana", "Bruno", "Carlos", "Daniel", "Mariana"]

    • Exiba apenas os dois primeiros convidados.
    • Exiba apenas os dois últimos convidados.
    • Exiba do segundo ao quarto convidado.

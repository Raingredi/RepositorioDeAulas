  2. Questão 2 — Prática: Divisão Segura

    Escreva um trecho de código Python que peça ao usuário dois números. O código deve tentar dividir o primeiro pelo segundo e exibir o resultado. Caso o usuário digite algo que não seja um número ou tente dividir por zero, o programa deve exibir uma mensagem de erro amigável, como "Entrada inválida. Por favor, insira apenas números e não divida por zero."

  3. Questão 3 — Identificação de Erro

    Um colega executa o código abaixo e recebe o erro ModuleNotFoundError: No module named 'numpy'. Qual é a causa provável do erro e qual comando ele deve executar no terminal para corrigi-lo?

    # Tenta importar a biblioteca para cálculos numéricos
import numpy as np

# Cria um array de números
dados = np.array([10, 20, 30])
print("Média:", np.mean(dados))

Projeto ENEM: Etapa 01

  1. Questão 1 — Comunicação

    Converse com a equipe de Python e JSON para alinhamento e responda às seguintes perguntas:

    • A - Quais dados são necessários para montar uma questão do ENEM?
    • B - O que é preciso para implementar um sistema de pontuação?
    • C - Quantas e quais tabelas serão necessárias para armazenar todos os dados?
    • D - Quais são as colunas de cada tabela e quais tipos de dados cada campo deve ter?

  2. Questão 2 — Relacionamento

    Com base nos dados definidos na Questão 1, responda:

    • A - Quais serão as PK (Primary Keys) e as FK (Foreign Keys)?
    • B - Quais serão os relacionamentos entre as tabelas?

  3. Questão 3 — Estruturação

    Usando a estrutura do MySQL e os dados coletados, descreva a definição completa das tabelas.

  4. Questão 4 — Consultas

    Após novo alinhamento com a equipe de Python e JSON, responda às seguintes perguntas e forneça os comandos SQL correspondentes:

    • A - Quais consultas a equipe de Python precisará para acessar as tabelas?
    • B - Quais consultas serão necessárias para inserir novas questões?
    • C - Quais consultas serão necessárias para atualizar a pontuação no ranking?

Atividade — Variáveis e Tipos de Dados em Python

  1. Questão 1 — Criando variáveis

    Crie variáveis em Python para representar as seguintes informações:

    • Seu nome (texto).
    • Sua idade (número inteiro).
    • Sua altura (número decimal).

    Mostre os valores com print().

  2. Questão 2 — Operações matemáticas

    Um aluno guardou 150 reais e ganhou mais 80. Depois comprou um fone de ouvido por 100.

    Crie variáveis para representar cada valor e calcule o quanto sobrou no final.

  3. Questão 3 — Concatenando texto e número

    Um jogador fez 120 pontos. Escreva um código que mostre a mensagem:

    "Sua pontuação é: 120"

    Dica: use str() ou f-strings (f"").

  4. Questão 4 — Calculadora simples

    Peça para o usuário digitar dois números (input()) e mostre a soma, subtração, multiplicação e divisão deles.

  5. Questão 5 — Corrigindo erro

    O seguinte código gera um erro:

    pontos = 50
    mensagem = "Você fez "
    resultado = mensagem + pontos
    print(resultado)

    Corrija o erro para que o programa mostre a frase corretamente.

  6. Questão 6 — Atividade prática

    No mercado, você comprou 3 bananas (R$ 1,20 cada) e 2 laranjas (R$ 2,00 cada).

    1. Crie variáveis para os preços das frutas.
    2. Calcule o valor total da compra.
    3. Mostre a mensagem: "O valor total é: R$X".

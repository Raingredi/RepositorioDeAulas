Atividade — Variáveis, Tipos e Operações em Python

Questões com exemplos diferentes da aula, mantendo os mesmos conceitos: variáveis, tipos, operações, conversão e depuração.

  1. Questão 1 — Conceito

    Explique com suas palavras o que é uma variável em programação e dê uma analogia diferente da aula (máximo duas frases).

  2. Questão 2 — Tipos básicos

    Dê um exemplo de cada tipo em Python usando contextos diferentes da aula (uma linha por tipo): int, float e str. Não repita os exemplos da aula.

  3. Questão 3 — Operações e tipos

    Calcule o resultado e indique o tipo em Python para cada expressão:

    • 7 + 4
    • 7 / 2
    • 7 * 2.5

  4. Questão 4 — Erro comum (TypeError)

    Explique por que cada um dos trechos abaixo gera erro e mostre uma correção simples para cada caso:

    # A
msg = "Total: "
valor = 50.0
print(msg + valor)

# B
quantidade = input("Qtd: ")
total = quantidade * 2
print(total)

  5. Questão 5 — f-strings

    Usando variáveis produto e valor, escreva uma f-string que imprima: \"O produto 'Caderno' custa R$12.50\". Mostre a linha de código.

  6. Questão 6 — Conversão de tipos

    Você recebeu do `input()` o preço como string: \"9.99\". Mostre como converter para float, multiplicar por 3 e armazenar em total (uma linha de código para a conversão e outra para o cálculo).

  7. Questão 7 — input() e validação básica

  8. Questão 8 — Conversão de temperatura

    Escreva as linhas principais que convertem temperatura em Celsius para Fahrenheit. Considere a variável c e calcule f com a fórmula f = c * 9/5 + 32. Mostre duas linhas: atribuição de c (exemplo) e cálculo de f.

  9. Questão 9 — Carrinho de compras

    Implemente a lógica para calcular o total de um carrinho com: 3 canetas a R$1,75 cada e 2 cadernos a R$6,40 cada. Escreva as linhas de definição das variáveis e do cálculo de `total` (3–4 linhas).

    # exemplo de resposta esperado
preco_caneta = 1.75
preco_caderno = 6.40
total = (3 * preco_caneta) + (2 * preco_caderno)
print(total)

plt.show()

  10. Questão 11 — Projeto Guiado: Comparando Vendas

    Você tem os meses ['Jan','Fev','Mar'] e vendas [100, 150, 120]. Indique rapidamente (em palavras ou código mínimo) como faria um gráfico para mostrar a evolução.

    Sugestão simples (apenas para entender):

    plt.plot(['Jan','Fev','Mar'], [100,150,120])
plt.title('Vendas por mês')
plt.ylabel('Vendas')
plt.show()

Atividade — Listas em Python

  1. Questão 1 — Criando listas

    Crie uma lista chamada animais com pelo menos 4 nomes de animais. Depois, imprima a lista completa na tela.

  2. Questão 2 — Acessando elementos

    Dada a lista cores = ["azul", "verde", "vermelho", "amarelo"]:

    • Mostre apenas a primeira cor.
    • Mostre a última cor.
    • Mostre a segunda cor da lista.

  3. Questão 3 — Alterando elementos

    Com a lista numeros = [5, 10, 15, 20]:

    • Troque o valor 10 por 99.
    • Imprima a lista atualizada.

  4. Questão 4 — Adicionando e removendo

    Usando a lista comidas = ["pizza", "hambúrguer", "sushi"]:

    • Adicione o item "lasanha" ao final da lista.
    • Remova o item "hambúrguer".
    • Mostre o resultado final na tela.

  5. Questão 5 — Tamanho da lista

    Crie uma lista chamada nomes com 5 nomes. Depois, use uma função do Python para mostrar quantos elementos a lista possui.

  6. Questão 6 — Juntando listas

    Crie duas listas:

    • lista1 = [1, 2, 3]
    • lista2 = [4, 5, 6]

    Depois, crie uma nova lista que contenha todos os números das duas.

  7. Questão 7 — Repetindo listas

    Crie uma lista chamada vogais = ["a", "e", "i", "o", "u"]. Mostre o resultado de repetir essa lista duas vezes usando o operador *.

  8. Questão 8 — Removendo tudo

    Crie uma lista com alguns números e depois use um método do Python para remover todos os elementos dela de uma vez. Mostre o resultado antes e depois de limpar a lista.

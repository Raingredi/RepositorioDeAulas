Módulo 1 — Algoritmos e Lógica de Programação

  1. Questão 1 — Descrevendo um Processo

    Descreva em uma sequência de passos lógicos (um algoritmo em linguagem natural) como você faria para sacar dinheiro em um caixa eletrônico.

    • Pense em todas as etapas, desde a inserção do cartão até a retirada do dinheiro.
    • Inclua pelo menos uma decisão (ex: "o saldo é suficiente?").

  2. Questão 2 — Identificando Componentes

    Considere o processo de fazer login em um site de e-mail. Identifique os três componentes essenciais do algoritmo:

    1. Entrada: Quais dados o usuário precisa fornecer?
    2. Processamento: O que o sistema faz com esses dados?
    3. Saída: Quais são os possíveis resultados que o usuário vê na tela?

  3. Questão 3 — Corrigindo a Ordem Lógica

    As instruções a seguir para fazer um sanduíche estão fora de ordem. Reorganize-as na sequência correta para que a lógica funcione.

    • Colocar o queijo e o presunto.
    • Pegar duas fatias de pão.
    • Juntar as duas fatias de pão.
    • Passar manteiga em uma das fatias.

  4. Questão 4 — Criando um Mini-Algoritmo

    Escreva um algoritmo simples em linguagem natural para decidir se você deve levar um guarda-chuva ao sair de casa. O algoritmo deve considerar se está chovendo ou se há previsão de chuva.

Módulo 2 — Fluxogramas e Pseudocódigo

  1. Questão 1 — Do Problema ao Fluxograma

    Desenhe um fluxograma simples que represente a lógica para verificar se um número é par ou ímpar.

    • O algoritmo deve receber um número como entrada.
    • O processamento principal é uma decisão baseada no resto da divisão por 2.
    • Deve haver duas saídas possíveis: "É par" ou "É ímpar".

  2. Questão 2 — Interpretando um Fluxograma

    Observe o fluxograma da aula (slide "Ferramenta de Planejamento: Fluxograma"). Descreva com suas palavras o que aquele algoritmo genérico está fazendo. Qual é o papel do losango (decisão) no fluxo?

  3. Questão 3 — Escrevendo Pseudocódigo

    Escreva um algoritmo em pseudocódigo (similar ao da aula) para calcular o troco de uma compra.

    1. Receba o valor_total_da_compra.
    2. Receba o valor_pago_pelo_cliente.
    3. Calcule o troco.
    4. Mostre o valor do troco na tela.

  4. Questão 4 — Traduzindo para Pseudocódigo

    Traduza a seguinte lógica para o formato de pseudocódigo:

    "Peça a idade do usuário. Se a idade for maior ou igual a 18, mostre a mensagem 'Acesso permitido'. Caso contrário, mostre 'Acesso negado'."

Módulo 3 — Operadores e Expressões Lógicas

    Considere que idade = 20 e altura = 1.75. Diga se cada uma das expressões abaixo resulta em True ou False.

    • idade >= 18
    • altura < 1.70
    • idade != 20
    • altura == 1.75

  2. Questão 2 — Combinando Condições com Operadores Lógicos

    • not e_filme_bom

    Para ter um desconto em uma loja, um cliente precisa ser VIP OU a compra deve ser maior que R$ 500,00. Crie variáveis e escreva uma única expressão lógica em Python que represente esta regra.

  4. Questão 4 — Ordem de Precedência

Módulo 4 — Estruturas de Controle

  1. Questão 1 — Qual Estrutura Usar?

    Para cada cenário abaixo, diga qual estrutura de controle (`if/else`, `for` ou `while`) é a mais apropriada e justifique sua escolha.

    • Cenário A: Exibir uma mensagem de "Bom dia", "Boa tarde" ou "Boa noite" dependendo da hora do dia.
    • Cenário B: Pedir para o usuário digitar uma senha repetidamente até que ele acerte.
    • Cenário C: Calcular a média de notas de uma lista com exatamente 10 alunos.

  2. Questão 2 — Teste de Mesa Manual

    O "teste de mesa" é uma técnica para simular a execução de um algoritmo no papel. Para o código Python abaixo, preencha uma tabela mostrando o valor da variável soma a cada passo do laço.

    soma = 0
    for numero in range(1, 5):
      soma = soma + numero
    print(soma)

  3. Questão 3 — Encontrando o Erro Lógico (Loop Infinito)

    O código abaixo cria um "loop infinito", ou seja, ele nunca para de executar. Por que isso acontece e como você poderia corrigi-lo?

    contador = 0
    while contador < 5:
      print("Olá!")

  4. Questão 4 — O que será impresso?

    Analise o código abaixo e determine exatamente o que será exibido na tela.

    temperatura = 28
    if temperatura > 25:
      print("Dia quente!")
    elif temperatura > 15:
      print("Dia agradável.")
    else:
      print("Dia frio.")
